# 小红书自动获客工具 - 打包说明

## 🚀 快速使用

### 方法1: 直接运行Python脚本 (推荐)

```bash
# 1. 安装依赖
pip install selenium undetected-chromedriver pandas openpyxl

# 2. 运行
python xiaohongshu_scraper_exe.py
```

### 方法2: 打包成exe

```bash
# 1. 安装PyInstaller
pip install pyinstaller

# 2. 打包
pyinstaller --onefile --noconsole --name "小红书获客工具" xiaohongshu_scraper_exe.py

# 3. exe文件在 dist/ 目录
```

## 📖 使用说明

### 1. 启动工具
- 双击运行 `xiaohongshu_scraper_exe.py`
- 或运行打包好的exe

### 2. 输入参数
```
请输入搜索关键词: 美国留学
请输入要爬取的笔记数量: 10
是否后台运行? (y/n): n
```

### 3. 登录小红书
- 浏览器会自动打开小红书
- **手动扫码登录** 或 **手机号登录**
- 登录成功后,按回车继续

### 4. 自动爬取
工具会自动:
1. 搜索关键词 "美国留学"
2. 获取前10个笔记
3. 进入每个笔记
4. 爬取评论区所有评论
5. 提取用户名、评论内容、用户主页链接
6. 筛选高意向评论 (包含"想咨询"、"求推荐"等关键词)
7. 保存到Excel

### 5. 查看结果
- `xiaohongshu_leads_20240221_143022.xlsx` - 所有评论
- `xiaohongshu_high_intent_20240221_143022.xlsx` - 高意向评论

## ⚠️ 注意事项

### 1. 登录方式
- **必须手动登录** (扫码或手机号)
- 不要用密码登录 (容易被检测)

### 2. 爬取速度
- 每个笔记间隔 3-6秒
- 避免被小红书检测
- 建议每次爬取 10-20个笔记

### 3. 账号安全
- 使用小号登录
- 不要频繁爬取
- 建议每天爬取 50-100个笔记

### 4. 数据使用
- 仅用于商业获客
- 不要滥用用户信息
- 遵守隐私保护法规

## 🔧 常见问题

### Q1: 浏览器打不开?
A: 安装Chrome浏览器,或更新到最新版本

### Q2: 找不到评论?
A: 小红书DOM结构可能变化,需要更新CSS选择器

### Q3: 被检测到自动化?
A: 使用undetected-chromedriver已经绕过大部分检测

### Q4: 爬取速度太慢?
A: 可以增加笔记数量,但注意账号安全
