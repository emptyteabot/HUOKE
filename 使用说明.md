## 🎯 核心功能

### 1. 信号驱动获客系统

**功能:** 自动检测8种留学行业买入信号

**使用方法:**
```python
from signal_driven_outreach import SignalDetector

detector = SignalDetector()

# 检测文本中的信号
text = "我在小红书搜索了英国留学申请,想了解UCL的CS专业"
signals = detector.detect_signals(text)

# 输出: [{'type': 'search_behavior', 'urgency': 'high', ...}]
```

**8种信号类型:**
- 搜索行为 (小红书/知乎搜索留学关键词)
- 咨询行为 (在论坛提问留学问题)
- 时间节点 (申请季/考试季临近)
- 竞品对比 (对比多家留学机构)
- 预算讨论 (讨论留学费用)
- 背景匹配 (GPA/语言成绩符合目标院校)
- 紧急需求 (急需文书修改/面试辅导)
- 决策信号 (准备签约/支付定金)

### 2. AI邮件生成系统

**功能:** 使用DeepSeek自动生成个性化邮件

**使用方法:**

4. **[CTA]** - "免费领取《UCL CS申请攻略》,微信扫码即可"


### Q4: 如何计算ROI?

**A:** 完整公式:
```python
# 投入成本
cost = (
    邮件发送费用 +  # SendGrid: ¥0.5/封
    DeepSeek API费用 +  # ¥0.01/次
    人工时间成本  # 假设¥100/小时
)

# 产出收益
revenue = (
    咨询人数 * 咨询转化率 * 客单价
)

# ROI
roi = (revenue - cost) / cost * 100

# 示例:
# 发送1000封邮件
cost = 1000 * 0.5 + 1000 * 0.01 + 2 * 100 = ¥710
# 20人咨询,30%转化,¥30,000客单价
revenue = 20 * 0.3 * 30000 = ¥180,000
# ROI
roi = (180000 - 710) / 710 * 100 = 25,246%
```

### Q6: 如何避免进入垃圾箱?

**A:** 6个关键点:
1. **SPF/DKIM验证** - 在SendGrid后台配置
2. **避免垃圾词** - 不用"免费""赚钱""点击这里"
3. **文本/HTML比例** - 保持1:1
4. **退订链接** - 必须包含
5. **发送频率** - 不要一次发太多
6. **用户互动** - 鼓励回复

```python
# 系统已内置反垃圾优化
sender.anti_spam_mode = True
```
