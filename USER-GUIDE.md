# 🎯 LeadPulse 使用指南

## 这是什么?

**LeadPulse** 是一个AI驱动的留学机构获客工具,帮助你:
- 📝 管理学生线索
- 🤖 AI自动生成个性化咨询邮件
- 📊 分析转化数据

---

## 🚀 快速开始

### 第1步: 添加学生

1. 点击首页的 **"学生管理"** 卡片
2. 填写学生信息:
   - 姓名、邮箱、电话
   - 目标国家、学历、专业
   - 预算范围
3. 点击 **"添加学生"**

### 第2步: AI生成邮件

1. 点击首页的 **"AI生成邮件"** 卡片
2. 选择刚才添加的学生
3. 选择邮件类型(首次咨询/留学规划/院校推荐等)
4. 填写你的机构信息
5. 点击 **"生成邮件"**
6. AI会自动生成个性化邮件!

### 第3步: 查看数据

1. 点击首页的 **"数据分析"** 卡片
2. 查看学生数量、邮件数量、转化率等

## 🔧 技术支持

### 常见问题

**Q: AI生成失败怎么办?**
A: 检查Secrets中的API Key是否正确配置

**Q: 数据会丢失吗?**
A: 数据存储在Streamlit云端,不会丢失

**Q: 可以导出数据吗?**
A: 可以,在学生列表页面点击"导出CSV"

**Q: 支持多少学生?**
A: 无限制
