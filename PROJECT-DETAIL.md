# 📋 GuestSeek 项目详细说明

## 🎯 这是什么?

**GuestSeek** 是一个**纯Web应用**,专为留学机构打造的AI获客系统。

**简单来说**:
- 打开网页就能用,不需要安装任何软件
- 像用ChatGPT一样简单
- 所有功能都在一个页面里

---

## 🌐 现在的架构

```
用户浏览器
    ↓
https://你的域名.streamlit.app (Web应用)
    ↓
├─ Supabase (数据库 - 云端)
├─ OpenAI API (AI生成邮件 - 云端)
└─ SendGrid API (发送邮件 - 云端)
```

**完全云端化,无需本地安装!**

---

## 📱 6个核心功能模块

### 1. 👥 学生管理
**干什么的**: 记录潜在客户信息
- 添加学生(姓名/邮箱/电话/目标国家/学历/专业/预算)
- 查看学生列表
- 编辑/删除学生

**使用场景**:
```
家长在小红书咨询 → 你添加到系统 → 保存到云端数据库
```

---

### 2. 🤖 AI生成邮件
**干什么的**: 用AI自动写个性化邮件
- 选择学生
- 选择邮件类型(首次咨询/留学规划/院校推荐/时间线/案例分享)
- 填写机构信息
- 点击生成 → AI自动写好邮件

**使用场景**:
```
选择"张三" → 选择"首次咨询邮件" → 点击生成 →
AI自动生成: "尊敬的张三家长,您好!关于您咨询的美国本科申请..."
```

**技术**: OpenAI GPT-4,3秒生成一封邮件

---

### 3. 📧 批量发送
**干什么的**: 一键给多个学生发邮件
- 勾选多个学生
- 写邮件模板(支持变量: {name}, {country}, {degree})
- 点击发送 → 自动给每个人发个性化邮件

**使用场景**:
```
勾选100个学生 → 写模板"尊敬的{name}家长,关于{country}留学..." →
点击发送 → 系统自动发100封个性化邮件
```

**技术**: SendGrid API,每秒可发10封

---

### 4. ⚡ 自动化工作流
**干什么的**: 设置规则,系统自动跟进
- 3天未打开邮件 → 自动发跟进邮件
- 7天未点击 → 自动发优惠邮件
- 高意向客户 → 自动提醒你

**使用场景**:
```
发了邮件给张三 → 3天后他没打开 → 系统自动发:
"张三家长,您好!几天前给您发的留学方案收到了吗?..."
```

**技术**: 定时任务引擎,每小时检查一次

---

### 5. 🌐 多平台获客
**干什么的**: 从LinkedIn/小红书/知乎找潜在客户
- 输入关键词(例如: "美国留学")
- 选择平台
- 点击搜索 → 找到潜在客户
- 批量导入到系统

**使用场景**:
```
搜索"美国留学" → 找到50个在小红书问留学的人 →
批量导入 → 给他们发邮件
```

**注意**: 目前是模拟数据,真实使用需要配置第三方服务

---

### 6. 📊 数据分析
**干什么的**: 看数据,算ROI
- **转化漏斗**: 100个线索 → 80个发邮件 → 40个打开 → 10个点击 → 2个转化
- **ROI计算**: 花了5000元 → 赚了20000元 → ROI = 300%
- **时间趋势**: 看每天发了多少邮件,打开率多少
- **客户分群**: 高意向/中意向/低意向客户分类
- **导出报表**: 下载Excel给老板看

**使用场景**:
```
月底了 → 打开数据分析 → 看到:
- 本月新增200个线索
- 发送500封邮件
- 打开率35%
- 转化5个客户
- ROI 250%
→ 导出报表给老板
```

## 🔧 技术架构

### 前端
- **Streamlit**: Python Web框架,写Python就能做网页
- **特点**: 简单、快速、适合数据应用

### 后端
- **Python**: 所有逻辑都是Python
- **无需单独后端**: Streamlit自带服务器

### 数据库
- **Supabase**: 云端PostgreSQL数据库
- **7张表**: users, leads, emails, templates, workflows, logs, scraping_jobs
- **安全**: 行级安全(RLS),每个用户只能看自己的数据

### AI
- **OpenAI GPT-4**: 生成邮件内容
- **你的API**: `sk-MRhl7sGPXCYtqtDx49fxuzv3SjbxrJlUza9ylZjlMTHYz0wu`
- **Base URL**: `https://oneapi.gemiaude.com/v1`

### 邮件

- **功能**: 发送邮件 + 追踪打开率/点击率
- **需要配置**: 注册SendGrid,获取API Key

## 🚀 如何使用?

### 方式1: 本地运行(测试用)
```bash
cd streamlit-app
pip install -r requirements.txt
streamlit run Home.py
```
打开浏览器访问: http://localhost:8501

### 方式2: 部署到云端(推荐)
1. 访问 https://streamlit.io/cloud
2. 连接GitHub仓库
3. 选择 `streamlit-app/Home.py`
4. 配置API密钥(在Secrets里)
5. 点击Deploy
6. 3分钟后得到网址: `https://你的应用.streamlit.app`

**任何人打开这个网址就能用!**
