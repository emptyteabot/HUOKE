# LeadPulse Reply SOP And Sample Packs

Use this only when a prospect replies with interest, confusion, or a request to see samples.

Hard rule: when possible, include real public Xiaohongshu account IDs or public post evidence in samples. Do not include phone numbers, WeChat IDs, private messages, or any non-public data. Keep samples tightly matched to the prospect's industry.

## Human Copy Rules

- Do not sound like a SaaS landing page or客服模板.
- First reply to the exact thing they said, then send only the minimum useful sample.
- Prefer "我" over "我们" unless a real team context is needed.
- Avoid repeated phrases: `高意向线索`, `精准获客`, `公开平台主动表达需求`, `脱敏样本`, `看看方向`, `是否对口`.
- Use concrete words from their business: `询盘不到10个`, `招生累`, `口语写作提分`, `找供应商`, `小B客户资源`.
- If the sample does not match their business, do not send it.
- If they are confused, explain in one sentence, not a pitch.


## If They Say "没太明白"

```text
我说简单点：不是卖课/代运营。我就是把已经在帖子里问“有没有靠谱老师/供应商/服务商”的人挑出来，给你原帖和开场角度。你先看两条，不准就算了。
```

## If They Ask "怎么收费"

Do not lead with price before samples. Reply:

```text
先别谈价格，先看两条质量。你觉得这种人真值得跟，再聊数量和交付；不准就到这。
```

## If They Ask "哪里来的数据"

```text
只看用户自己发出来的帖子/评论，不碰手机号、微信、私信。每条都会给原帖和我为什么觉得能跟。
```
