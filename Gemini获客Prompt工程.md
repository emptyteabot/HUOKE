## 📋 完整Prompt模板

### Prompt 1: 寻找目标客户聚集地

```
我需要找到正在寻找留学服务的潜在客户。请帮我:

1. 列出中国学生讨论留学申请的TOP 20平台/社区/论坛
   - 包括:小红书、知乎、豆瓣、微博、留学论坛等
   - 标注每个平台的活跃度(高/中/低)
   - 标注是否容易获取联系方式

2. 对于每个平台,给出:
   - 最活跃的留学话题/标签(至少5个)
   - 典型的高意向用户行为特征
   - 如何识别"正在决策期"的用户

3. 提供具体的搜索关键词组合(至少20组)
   例如:"英国留学 求推荐中介"
   例如:"美国研究生申请 DIY还是找机构"

4. 给出数据抓取策略:
   - 哪些平台可以直接看到用户联系方式
   - 哪些平台需要私信才能要到联系方式
   - 哪些平台有公开的邮箱/微信

请以表格形式输出,方便我直接使用。
```

**预期输出:**
- 20个平台列表
- 100+个搜索关键词
- 每个平台的获客策略

### Prompt 7: 综合分析与优先级排序

```
我已经从多个平台收集了客户数据。请帮我:

**任务1: 数据去重**
检查以下数据源,找出重复的用户(同一个人在多个平台出现):
- 小红书用户50个
- 知乎用户30个
- 豆瓣用户20个
- 微博用户25个
- 论坛用户40个

去重逻辑:
- 相同的联系方式(微信/邮箱/QQ)
- 相似的背景描述(学校/专业/目标)
- 相似的用户名

**任务2: 意向评分**
对所有用户重新评分(1-10分),综合考虑:
- 平台数量(出现在越多平台,意向越强)
- 行为频率(发帖/评论越频繁,意向越强)
- 时间紧迫度(申请季临近,分数越高)
- 预算匹配度(预算>20万,分数越高)
- 联系方式完整度(有邮箱/微信,分数+1)

**任务3: 优先级排序**
将所有用户分为3档:
- S级(评分9-10): 立即联系,24小时内
- A级(评分7-8): 3天内联系
- B级(评分5-6): 1周内联系

**任务4: 生成触达计划**
对于每个用户,生成:
- 最佳触达时间(基于他们的活跃时间)
- 最佳触达渠道(私信/邮件/微信)
- 个性化开场白(提到他们的具体问题)
- 价值钩子(免费资料/免费咨询/成功案例)

以Excel格式输出完整的客户数据库:
```
Sheet1: S级客户(立即联系)
Sheet2: A级客户(3天内)
Sheet3: B级客户(1周内)

每个Sheet包含:
姓名,平台,联系方式,背景,目标,预算,意向评分,触达时间,触达渠道,开场白,价值钩子,备注
```

请给我一个完整的、可执行的触达计划。
```

**预期输出:**
- 去重后的完整客户数据库
- 按优先级排序
- 每个客户的触达计划
