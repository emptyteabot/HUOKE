# 知乎爬虫 - 快速开始

## 一键启动

### Windows用户
双击运行: `启动知乎爬虫.bat`

### 手动启动

1. 安装依赖:
```bash
pip install -r requirements.txt
```

2. 运行爬虫:
```bash
python zhihu_scraper.py
```

## 使用流程

```
1. 程序启动 → 打开Chrome浏览器
2. 手动登录 → 扫码或密码登录知乎
3. 输入关键词 → 例如: "Python爬虫"
4. 设置数量 → 例如: 10 (爬取10个回答)
5. 自动爬取 → 等待完成
6. 导出Excel → 自动保存到当前目录
```

## 输出示例

```
知乎数据_20260222_143025.xlsx
├── 类型: 回答/评论
```

## 核心特性

✅ **绕过反爬**: 使用 undetected-chromedriver
✅ **安全登录**: 手动登录,不保存密码
✅ **智能爬取**: 自动滚动加载更多内容
✅ **随机延迟**: 1-5秒随机延迟,避免封号
✅ **完整数据**: 回答+评论+用户信息
✅ **Excel导出**: 一键导出,方便分析

## 注意事项

⚠️ 首次运行会自动下载 chromedriver (需要几分钟)
⚠️ 确保Chrome浏览器已安装
⚠️ 不要设置过大的爬取数量 (建议≤50)
⚠️ 遵守知乎使用条款,仅用于学习研究

## 常见问题

**Q: 提示找不到Chrome?**
A: 安装Chrome浏览器: https://www.google.com/chrome/

**Q: 爬取速度慢?**
A: 这是正常的,随机延迟是为了避免被封

**Q: 数据为空?**
A: 检查是否登录成功,或者更换关键词

**Q: 程序卡住?**
A: 按 Ctrl+C 中断,检查浏览器是否有验证码
