# 给Gemini的项目说明

## 项目概述

这是一个**AI驱动的留学获客系统**,核心目标是帮助留学中介机构自动化获取潜在客户。

## 当前状态

### ✅ 已完成的功能

1. **邮件营销系统** (email_auto_sender.py)
   - 集成DeepSeek API生成个性化邮件
   - 支持多触点序列(Day 1/3/7/14)
   - A/B测试框架
   - 效果追踪(打开率/回复率)

2. **信号驱动系统** (intent_signal_hijacker.py等)
   - 检测8种留学行业买入信号
   - 基于信号生成破防式文案
   - 量化淘汰机制(200次触达止损线)

3. **AI评分系统** (streamlit-app/lead_scoring.py)
   - 自动评估客户意向(1-10分)
   - 优先级分类(S/A/B级)

4. **爬虫系统** (scrapers/)
   - 小红书/知乎/微博爬虫
   - 但有问题:需要手动登录,反爬严重,抓不到联系方式

### ❌ 核心问题

**最大的问题:缺少真实客户数据源**

- 现有的`ai_lead_generator.py`生成的是假数据(AI编造的)
- 爬虫虽然能抓数据,但抓不到联系方式(邮箱/手机号)
- 没有真实客户数据,整个系统就是空中楼阁

## 解决方案:用Gemini做获客第一步

### 为什么选择Gemini?

1. **Deep Research功能** - 能访问实时网页数据
2. **Google搜索集成** - 能搜索所有公开平台
3. **AI分析能力** - 能从文本中提取联系方式和意向
4. **结构化输出** - 能直接生成JSON/CSV格式

### 我需要Gemini做什么?

**核心任务:从中国的社交平台找到正在寻找留学服务的潜在客户**

具体步骤:

#### 第1步: 找到客户聚集地
搜索并列出中国学生讨论留学申请的平台:
- 小红书 (xiaohongshu.com)
- 知乎 (zhihu.com)
- 豆瓣小组 (douban.com)
- 微博 (weibo.com)
- 留学论坛(寄托天下、一亩三分地等)

#### 第2步: 搜索高价值内容
在这些平台搜索关键词:
- "留学中介推荐"
- "英国留学申请"
- "DIY申请求助"
- "留学机构避坑"
- "求推荐靠谱中介"

#### 第3步: 提取用户信息
对于每个发帖/评论的用户,提取:
- 用户名/昵称
- 用户主页链接
- 联系方式(如果公开了邮箱/微信/QQ)
- 背景信息(从发帖内容推断:学校/专业/GPA)
- 目标信息(目标国家/学校/专业)
- 预算范围
- 痛点和需求

#### 第4步: 意向评分
对每个用户打分(1-10分):
- 发帖频率(越频繁越高)
- 提问具体度(越具体越高)
- 时间紧迫度(申请季临近越高)
- 预算匹配度(预算>20万越高)

#### 第5步: 输出标准化数据
以JSON格式输出:
```json
[
  {
    "name": "张同学",
    "platform": "小红书",
    "profile_url": "https://...",
    "contact": {
      "email": "zhang@qq.com",
      "wechat": "zhang123",
      "phone": "13812345678"
    },
    "background": {
      "school": "某985大学",
      "major": "计算机科学",
      "gpa": "3.5"
    },
    "target": {
      "country": "英国",
      "university": "UCL",
      "major": "CS",
      "degree": "硕士"
    },
    "budget": "30-50万",
    "pain_points": ["不知道如何选校", "文书不会写"],
    "signals": ["在小红书搜索了UCL申请", "评论区咨询中介"],
    "intent_score": 8,
    "priority": "S",
    "notes": "最近7天内多次发帖,非常紧迫"
  }
]
```

### 预期产出

- **数量**: 150-200个潜在客户
- **质量**: 意向评分>7分的占60%以上
- **联系方式完整率**: 60-70%(有邮箱或微信)
- **时间**: 2-3小时

### 数据导入

Gemini返回数据后,我会用`gemini_data_importer.py`导入系统:
```bash
python gemini_data_importer.py --input gemini_leads.json --output leadpulse_leads.json
```

然后就可以开始发送邮件了!
